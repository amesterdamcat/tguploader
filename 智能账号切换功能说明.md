# 智能账号切换功能说明

## 功能概述

全新升级的账号切换功能，支持快速切换和可选的重新配置，大大简化了多账号管理流程。

## 核心特性

### 🚀 智能切换模式
- **快速切换**: 使用缓存配置，无需重新输入API信息
- **重新配置**: 更新API信息或频道设置  
- **只改频道**: 保持API配置不变，仅更新频道

### 💾 配置缓存系统
- 自动保存账号配置到本地JSON文件
- 支持多账号配置缓存和管理
- 按最后使用时间智能排序

## 使用流程

### 原有流程 vs 新流程对比

**原有繁琐流程:**
```
1. 选择切换账号
2. 重新输入API ID ❌ 
3. 重新输入API Hash ❌
4. 重新输入频道ID
5. 验证和登录
```

**新的智能流程:**
```
🚀 快速切换:
1. 选择切换账号  
2. 选择"快速切换"
3. 一键完成！✨

⚙️ 重新配置 (仅需要时):
1. 选择切换账号
2. 选择"重新配置" 
3. 输入新的API信息
4. 自动保存到缓存

📺 只改频道:
1. 选择切换账号
2. 选择"只更改频道"
3. 输入新频道ID
4. 完成！
```

## 技术架构

### 配置缓存结构
```json
{
  "phone": "+8613812345678",
  "api_id": "1234567",
  "api_hash": "abcdef1234567890abcdef1234567890",
  "channel_id": "MyChannel",
  "created_time": "2025-09-13T10:30:45.123456",
  "last_used": "2025-09-13T11:00:32.654321"
}
```

### 文件存储
- 位置: `.account_configs/` 目录
- 命名: `{phone_number}.json` (去掉特殊符号)
- 编码: UTF-8
- 格式: JSON

### 核心方法

#### `save_account_config(phone, api_id, api_hash, channel_id)`
- 保存账号配置到本地缓存
- 自动创建配置目录
- 更新时间戳信息

#### `load_account_config(phone)`
- 加载指定账号的配置
- 自动更新最后使用时间
- 返回完整配置信息

#### `list_cached_accounts()`
- 列出所有缓存的账号
- 按最后使用时间排序
- 包含完整配置信息

#### `delete_account_config(phone)`
- 删除指定账号的配置缓存
- 安全的文件删除操作
